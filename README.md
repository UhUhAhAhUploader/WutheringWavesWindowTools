# WutheringWavesWindowTools

主要目的是为鸣潮游戏添加一个黑色背景框

# 开始

脚本`wuwa_run.ps1`, 位于仓库的根目录

# 演示

启动：
`powershell -ExecutionPolicy Bypass -Command ".\wuwa_run.ps1"`

带参数启动：
`powershell -ExecutionPolicy Bypass -Command ".\wuwa_run.ps1 'D:\Wuthering Waves\Wuthering Waves Game\Client\Binaries\Win64\Client-Win64-Shipping.exe' '鸣潮  ' '-windowed -ResX=1920 -ResY=700'"`
